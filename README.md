# The IceHawk framework

Fast and reliable PHP frontend framework respecting CQRS.

## Requirements

 * PHP >= 5.5
 * xdebug extention to run the tests

## Installation

Add this to your `composer.json`:

```json
require: {
    "fortuneglobe/icehawk": "1.3.0"
}
```
